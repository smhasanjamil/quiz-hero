Question 1: Difference between map, filter, forEach and find.
Function Description Return Value
map() Executes the given function on each element of an array and returns a new array with the updated values. A new array with the updated values.
filter() Creates a new array with all elements that pass the test implemented by the provided function. A new array with the filtered elements.
forEach() Executes the provided function once for each array element. It does not return anything. Undefined
find() Returns the first element in an array that satisfies the provided testing function. The first element that satisfies the condition or 'undefined' if no such element is found.
