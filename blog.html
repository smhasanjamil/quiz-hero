Question 1: Difference between map, filter, forEach and find.
Function Description Return Value
map() Executes the given function on each element of an array and returns a new array with the updated values. A new array with the updated values.
filter() Creates a new array with all elements that pass the test implemented by the provided function. A new array with the filtered elements.
forEach() Executes the provided function once for each array element. It does not return anything. Undefined
find() Returns the first element in an array that satisfies the provided testing function. The first element that satisfies the condition or 'undefined' if no such element is found.

Question 3: What is event loop and how does it work?

The event loop is a key feature of JavaScript that enables it to handle asynchronous operations efficiently.

The Event Loop has one simple job, to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.

Question 4: Purpose of 'quizTimer()' function

The 'quizTimer()' function is responsible for handling the timer functionality for the quiz. It takes a single parameter 'dismiss', which is a boolean value. If 'dismiss' is true, the function clears the timer by calling 'clearInterval(timer)', and returns. If 'dismiss' is false, the function sets an interval using 'setInterval()' that executes a callback function every second.

Question 5: Purpose of 'chooseQuiz()' function

It takes two arguments: the index of the quiz and the answer given by the user. The function first checks if the given quiz exists in the answers array. If it exists, it updates the answer for that quiz. Otherwise, it adds the new answer to the answers array. Finally, the displayAnswers() function is called to update.
